# Twitta

https://aa-twitta.herokuapp.com/

Twitta is a Twitter clone that has a fair amount of functionality.

Features

- Tweets
- Comments
- Notifications
- Bookmarks
- Follow Suggestions
- Messaging (WebSockets)

Future Features

- Light / Dark mode
- Tags
- Mentioning users in tweets
- Infinite Scroll
- Explore Page

Stack

- React
- Flask
- PostgreSQL
